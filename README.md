JWT Angular Authentication

This project is an Angular frontend application that implements JWT authentication by communicating with a backend API for secure login, token storage, and protected routes.

📖 About

An Angular frontend application that uses JWT authentication to securely handle user login, token management, and protected API access.

🚀 Features
User login & authentication
JWT token handling
Protected routes (Route Guards)
HTTP interceptor for token injection
Secure API communication
Logout functionality
Role-based access support (optional)
